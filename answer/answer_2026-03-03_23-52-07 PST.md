## Notes

1. **Application Insights** — No dedicated resource file exists in the `resources/` folder. Application Insights (`Microsoft.Insights/components`) is created as a linked resource by AI Foundry Hub. Use `Monitoring Contributor` at the Resource Group scope. It requires a Log Analytics Workspace as its backing store.

2. **Azure DNS Private Resolver** — No dedicated resource file exists. Per the Private DNS Zones file, deploying a DNS Private Resolver (`Microsoft.Network/dnsResolvers`) requires **`Network Contributor`** at the Resource Group scope.

3. **Private Endpoint** — No dedicated resource file exists. Private Endpoints are covered across multiple resource files. Creating a Private Endpoint requires **`Network Contributor`** on the subnet plus the resource-specific contributor role on the target resource.

4. **Foundry IQ** is in **Public Preview** (no SLA) — role names may change before GA. The Foundry project's managed identity requires `Search Index Data Contributor` (not Reader) on AI Search because agentic retrieval dynamically creates and manages index structures.

5. **Microsoft AI Foundry Hub** requires `Contributor` (no narrower built-in role exists) and automatically provisions linked resources: Storage Account, Key Vault, and optionally Container Registry and Application Insights.

6. **Management vs. Data Plane separation** applies to: Azure OpenAI, AI Services, AI Search, Key Vault, Cosmos DB, and Storage Account. Management-plane roles (e.g., `Cognitive Services Contributor`) do **not** grant data-plane access and vice versa.

7. **Cosmos DB data-plane RBAC** uses Cosmos DB built-in roles (`Cosmos DB Built-in Data Contributor` / `Cosmos DB Built-in Data Reader`) assigned via ARM/CLI, separate from Azure RBAC. Assign `Cosmos DB Built-in Data Contributor` to your application's managed identity for document CRUD.

8. **Disable API key authentication** (`disableLocalAuth: true`) on Azure OpenAI, AI Services, Document Intelligence, AI Search, and Key Vault — use Entra ID token-based auth in production.
